3D Lunar Orbit System (Unreal Engine 5 – Blueprints)

This project demonstrates a real-time Earth–Moon orbital system built in Unreal Engine 5 using Blueprint scripting.

Features

Moon orbiting Earth using hierarchical transforms

Independent axial rotation for both bodies

Adjustable orbital and rotational speed parameters

Modular actor setup for expansion

Technical Focus

Blueprint-based transform manipulation

Parent-child component relationships

Real-time update logic

Clean actor structuring within UE5

In Progress

Expanding toward more physically accurate orbital behavior

Parameter-driven orbital scaling
